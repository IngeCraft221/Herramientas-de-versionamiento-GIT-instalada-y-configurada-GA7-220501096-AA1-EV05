## 📌 🧰 Comandos utilizados en Git

### ⚙️ Configuración inicial

git config --global user.name "TuNombre"
git config --global user.email "[tu@email.com](mailto:tu@email.com)"

---

### 📂 Inicialización del repositorio

git init

---

### 🔍 Verificación del estado

git status

---

### ➕ Agregar archivos

git add .
git add archivoprueba.txt

---

### 💾 Guardar cambios (commit)

git commit -m "Se realiza prueba inicial del repositorio"

---

### 🌿 Manejo de ramas

git branch
git checkout -b nueva-rama
git checkout proyectos

---

### ✏️ Modificación de archivos

nano archivoprueba.txt

---

### 👀 Ver contenido del archivo

cat archivoprueba.txt

---

### 🔄 Ver cambios realizados

git diff

---

### 🔗 Conexión con repositorio remoto

git remote add origin [https://github.com/usuario/repositorio.git](https://github.com/usuario/repositorio.git)
git remote -v

---

### ☁️ Subir cambios a GitHub

git push -u origin main
git push
git push origin proyectos

---

### ⬇️ Actualizar repositorio

git pull

---

### 🔀 Unión de ramas (merge)

git checkout main
git merge proyectos
git push
